# 公开开发跟踪

MoonChartKit 按公开仓库持续开发方式推进，核心证据包括提交记录、Issue、Pull Request、CHANGELOG、ROADMAP 和 CI。

## 当前提交主题

- 基础骨架与仓库元数据
- 主题系统
- 图表布局与数据统计
- SVG 画布基础设施
- 柱状图 SVG
- 折线图 SVG
- Sparkline
- Metric Card
- CLI SVG 演示
- CI 与协作模板
- README、路线图和更新日志

## 后续工单建议
